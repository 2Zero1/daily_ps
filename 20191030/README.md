# 문제1

> 주어진 정수를 문자열로 변환한다.
```
ex) 1234 => '1234'
```

## 1. 이해

- 모르는 것: 숫자의 각 자릿수를 ex)1, 2, 3, 4처럼 따로 분리할 수 있어야 한다. 근데 0은 어쩌지?
- 알고 있는 것: 정수이다.

## 2. 계획

1. 각 자릿수를 분리하면 `10^n * 정수` 인 정수로 분리가능하다. 즉 10^n으로 나눈다면 정수만 분리할 수 있다.
2. 가장 높은 자릿수부터 1을 반복하여 나온 결과를 순서대로 나열한다.
